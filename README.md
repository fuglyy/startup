# startup

Идея №1
Проблема:
Тысячи студентов ежегодно не получают место в общежитии и сталкиваются с проблемой поиска доступного жилья. Аренда квартиры в одиночку непосильна для бюджета, а поиск соседа по объявлениям в соцсетях — это рискованно, долго и неудобно.

Решение:
Сервис по автоматическому подбору сожителей для студентов. Мы помогаем найти идеального соседа по комнате или квартире на основе совпадения интересов, привычек и личных предпочтений, избавляя от необходимости просматривать сотни объявлений и договариваться с незнакомцами впустую.


Идея №2 - Платформа управления жизнью общежития
Проблема
Жизнь студентов в общежитии часто организована через большие чаты в Telegram, разбитые по темам (прачечная, объявления, досуг, важная информация). Со временем такие чаты превращаются в хаос: сообщения теряются, важная информация дублируется, сложно отслеживать загрузку общих зон (прачечная, кухни), возникают конфликты из-за бытовых мелочей и тратится лишнее время на организационные вопросы.

Решение
Создание специализированной платформы для общежитий — «цифровой инфраструктуры» студенческой жизни. Сервис объединяет объявления, маркетплейс вещей, события, бытовые сервисы, уведомления и структурированную важную информацию в одном интерфейсе. Это снижает хаос коммуникации, экономит время студентов и упрощает управление общежитием.
